# คำอธิบายโปรเจกต์ aaitaam-ex-app

## ภาพรวม

โปรเจกต์นี้เป็นหน้าเว็บแอปพลิเคชัน (Single-Page Application) ขนาดเล็กที่สร้างขึ้นด้วย React, Vite และ Tailwind CSS มีจุดประสงค์เพื่อใช้เป็นไมโครไซต์หลายภาษาสำหรับนิทรรศการ 'Maison du Temple: Thub Thab'

## สถาปัตยกรรม

สถาปัตยกรรมของโปรเจกต์นี้เรียบง่ายและเข้าใจได้ง่าย:

1.  **`App.jsx`**: เป็นคอมโพเนนต์หลักที่ควบคุมการทำงานทั้งหมดของแอปพลิเคชัน โดยมี state ที่ชื่อว่า `selectedLang` เพื่อเก็บภาษาที่ผู้ใช้เลือก
2.  **หน้า Landing**: หากยังไม่มีการเลือกภาษา (`selectedLang` เป็นค่าว่าง) แอปจะแสดงหน้า Landing ซึ่งดึงรายการภาษามาจากไฟล์ `data/lang_code.json`
3.  **หน้า ContentScreen**: เมื่อผู้ใช้เลือกภาษา state `selectedLang` จะถูกอัปเดต และแอปจะแสดงผลหน้า `ContentScreen`
4.  **การแสดงเนื้อหา**: `ContentScreen` จะดึงข้อความที่ตรงกับภาษาที่เลือกจากไฟล์ `data/content.json` มาแสดงผล นอกจากนี้ยังมีปุ่ม 'ย้อนกลับ' เพื่อรีเซ็ตค่า state และกลับไปยังหน้าเลือกภาษา

## ไฟล์สำคัญ

*   **`src/App.jsx`**: เป็นไฟล์ที่สำคัญที่สุด ประกอบด้วยโค้ดสำหรับ UI, การจัดการ state, และคอมโพเนนต์ทั้งหมดของแอป (`App`, `Landing`, `ContentScreen`)
*   **`data/content.json`**: ทำหน้าที่เหมือนฐานข้อมูลของโปรเจกต์ ใช้เก็บเนื้อหาและคำแปลทั้งหมดสำหรับนิทรรศการ
*   **`data/lang_code.json`**: ใช้สำหรับแสดงรายการภาษาที่มีให้เลือกในหน้า Landing
*   **`package.json`**: ระบุข้อมูลของโปรเจกต์, เทคโนโลยีที่ใช้, และ dependency ต่างๆ
