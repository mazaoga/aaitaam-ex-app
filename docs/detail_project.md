# คำอธิบายโปรเจกต์ aaitaam-ex-app (ฉบับปรับปรุง)

## ภาพรวม

โปรเจกต์นี้เป็นหน้าเว็บแอปพลิเคชัน (Single-Page Application) ขนาดเล็กที่สร้างขึ้นด้วย React, Vite และ Tailwind CSS มีจุดประสงค์เพื่อใช้เป็นไมโครไซต์หลายภาษาสำหรับนิทรรศการ 'Maison du Temple: Thub Thab'

## สถาปัตยกรรม (Architecture)

สถาปัตยกรรมของโปรเจกต์ได้ถูกปรับปรุงใหม่ทั้งหมดเพื่อรองรับการทำงานตาม URL (URL-based Routing) โดยใช้ `react-router-dom` เป็นส่วนสำคัญ

### โครงสร้างการทำงาน (Flow)

1.  **การเข้าถึงหน้าเว็บ**: เมื่อผู้ใช้เข้าสู่เว็บไซต์ผ่าน URL เช่น `/introduce`, `/room1` โปรเจกต์จะใช้ URL parameter (`contentKey`) เพื่อแสดงหน้า **เลือกภาษา** สำหรับเนื้อหาส่วนนั้นๆ
2.  **การเลือกภาษา**: ผู้ใช้เลือกภาษาที่ต้องการจากหน้าเลือกภาษา
3.  **การแสดงเนื้อหา**: หลังจากเลือกภาษา แอปพลิเคชันจะนำทางผู้ใช้ไปยัง URL ใหม่ในรูปแบบ `/{contentKey}/{lang}` (เช่น `/introduce/TH`, `/room1/EN`) เพื่อแสดงเนื้อหาที่ถูกต้องตามที่ URL ระบุ
4.  **การย้อนกลับ**: ผู้ใช้สามารถกดปุ่มย้อนกลับเพื่อกลับไปยังหน้าเลือกภาษาของเนื้อหาส่วนนั้นๆ ได้

### คอมโพเนนต์หลักใน `App.jsx`

*   `LanguageSelectorPage`: คอมโพเนนต์ที่ทำหน้าที่เป็นหน้าสำหรับให้ผู้ใช้เลือกภาษา โดยจะแสดงผลเมื่อเข้าถึง URL `/:contentKey`
*   `ContentDisplayPage`: คอมโพเนentสำหรับแสดงเนื้อหาตามภาษาและประเภทของเนื้อหาที่ระบุใน URL (`/:contentKey/:lang`)

## การเปลี่ยนแปลงล่าสุด

### 1. การจัดการ Routing
- เพิ่ม dependency `react-router-dom` เพื่อจัดการ Client-Side Routing
- เปลี่ยนโครงสร้าง `App.jsx` จากเดิมที่ใช้ State (`selectedLang`) ในการสลับหน้า มาเป็นการใช้ระบบ Route ตาม URL ทั้งหมด

### 2. การอัปเดตเนื้อหา
- เพิ่มข้อมูลจำลองสำหรับ `room3`, `room4`, และ `room5` ในไฟล์ `data/content.json`

### 3. การตั้งค่าโปรเจกต์และการ Deploy
- **`.gitignore`**: สร้างไฟล์ `.gitignore` เพื่อให้ Git ละเว้นไฟล์และโฟลเดอร์ที่ไม่จำเป็น (เช่น `node_modules`, `dist`) ซึ่งช่วยให้ Repository มีขนาดเล็กลงและสะอาดขึ้น
- **`netlify.toml`**: สร้างไฟล์ตั้งค่าสำหรับ Netlify เพื่อแก้ปัญหา "Page Not Found (404)" บนเว็บที่ deploy แล้ว โดยไฟล์นี้จะสั่งให้ Netlify ส่ง `index.html` กลับไปสำหรับทุก request เพื่อให้ React Router ทำงานได้อย่างถูกต้อง

## ไฟล์สำคัญ

*   **`src/App.jsx`**: เป็นไฟล์หลักที่กำหนดโครงสร้างของ Routes และจัดการการแสดงผลของแต่ละหน้า
*   **`data/content.json`**: ฐานข้อมูลของโปรเจกต์ ใช้เก็บเนื้อหาและคำแปลทั้งหมด
*   **`data/lang_code.json`**: ใช้สำหรับแสดงรายการภาษาที่มีให้เลือก
*   **`package.json`**: ระบุข้อมูลของโปรเจกต์และ dependencies ต่างๆ (รวมถึง `react-router-dom`)
*   **`.gitignore`**: ไฟล์สำหรับระบุให้ Git ละเว้นไฟล์ที่ไม่ต้องการติดตาม
*   **`netlify.toml`**: ไฟล์ตั้งค่าสำหรับ deploy บน Netlify
